# Multiple-Trade-Analyzer

Description: 
This is the first update to my trade calculator. 

The program asks the user how many trades they want to analyze. The program asks the user how many trades they want to analyze, then collects inputs for each trade (pair, entry, stop loss, and take profit).

It calculates:

1.Risk
2.Reward
3. Risk-to-reward ratio (RRR)
4. Trade quality (good, average, or avoid)

At the end, it compiles and displays the number of trades in each category.

What I practiced:
1. Loops
2. Conditionals (if/elif/else)
3. f-strings
4. Tracking trade performance

I am still reinforcing these concepts, but this project helped connecting them.

I struggled with:
1. Understanding the difference between continue (used in loops to skip invalid trades) and return (used in functions to stop execution)
2. Compiling the quality of trades at the end of the program using counters
3. If and while loops - not fully grasped but I managed to use them correctly.
