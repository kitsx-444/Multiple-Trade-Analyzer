# Multiple-Trade-Analyzer

Description: 
This is the first update to my trade calculator. The program will first require you to input the amount of trades you want to analyze. It will give you input questions for each of the trades you want to analyze. Afterward it will calculate your price level inputs and give you the data of your trades risk, reward, ratio and quality of each of them. The program will then compile the quanitity of the trade qualities at the bottom.

What I practiced
1. Loops
2. Conditionals
3. f-strings
4. Tracking trade performance

I am still reinforcing these concepts, but this project helped connecting them.
